# Estudos de Desenvolvimento Web com HTML e CSS 

 Este repositório destina-se ao armazenamento dos meus estudos e práticas relacionadas ao desenvolvimento web. Abaixo, você encontrará uma lista dos principais conceitos que explorei:

 ## Conteúdos de Estudo:

### HTML:

- Estrutura básica de HTML
- Utilização de headings para hierarquia de títulos
- Semântica HTML para melhorar a compreensão do conteúdo

### CSS - Box Model:

- Compreensão e aplicação do Box Model
- Manipulação de elementos usando o Box Sizing
- Estilização de listas e tabelas

### Seletores e Estilização Avançada:

- Utilização de seletores para direcionar estilos específicos
- Compreensão de pseudo-classes e pseudo-elementos
- Entendimento da especificidade para resolver conflitos de estilos

### Posicionamento e Layout:

- Trabalho com position e position relative
- Uso de unidades de medidas para garantir um design responsivo
- Media Queries para adaptação a diferentes dispositivos

### Efeitos Visuais:

- Arredondamento de bordas (border-radius)
- Criação de sombras com box-shadow
- Formatação avançada de texto

### Layouts Avançados:

- Flexbox para criação de layouts flexíveis
- Grid para estruturas mais complexas

## Como Navegar neste Repositório:

Cada diretório neste repositório aborda um tópico específico. Por exemplo:

- **/html-basics:** Conceitos fundamentais de HTML
- **/css-box-model:** Aplicações práticas do Box Model em CSS
- **/advanced-selectors:** Utilização de seletores avançados em CSS
- ...

Sinta-se à vontade para explorar e revisitar qualquer tópico de interesse. Este repositório é uma compilação do meu aprendizado contínuo e práticas no desenvolvimento web. Se tiver alguma dúvida ou sugestão, não hesite em entrar em contato!
